## **Abstract**

Biodiversity conservation is a critical global challenge addressed under the United Nations Sustainable Development Goal (SDG) 15 â€“ Life on Land. Accurate and timely identification of animal species plays a vital role in wildlife monitoring, conservation planning, and ecological research. However, traditional species identification methods are manual, time-consuming, and heavily dependent on domain experts. This project presents an AI-driven biodiversity monitoring system that leverages deep learning and computer vision to automatically classify animal species from images. Using a Convolutional Neural Network (CNN) with transfer learning, the system provides reliable species predictions along with confidence scores. The solution is deployed as an interactive web application, enabling easy access for researchers, students, and conservation stakeholders. ## **5. Methodology**

### **5.1 Model Architecture**

A Convolutional Neural Network (CNN) with **transfer learning** using **MobileNetV2** is employed. Transfer learning allows the model to leverage pre-trained knowledge from large-scale image datasets, improving accuracy and reducing training time.

### **5.2 Image Preprocessing**

* Image resizing to 224Ã—224 pixels
* Normalization of pixel values
* Batch processing for efficient training

### **5.3 Training and Evaluation**

The model is trained using categorical cross-entropy loss and optimized using the Adam optimizer. Performance is evaluated using accuracy metrics, achieving strong classification results across supported species.

### **5.4 Prediction Strategy**

The deployed system provides:

* Top-3 species predictions
* Confidence scores for each prediction
* Confidence-based interpretation for reliability assessment

---

## **6. System Architecture**

1. User uploads an animal image via the web interface
2. Image is preprocessed and passed to the trained CNN model
3. Model generates species predictions with confidence scores
4. Results are displayed through an interactive Streamlit interface

The architecture ensures modularity, scalability, and ease of deployment.
